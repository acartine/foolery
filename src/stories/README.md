# stories

Storybook stories for visual component development and review.

## Key Files

- **`beat-table.stories.tsx`** — Beat table with mock data in various states
- **`beat-form.stories.tsx`** — Beat creation/edit form variants
- **`beat-status-badge.stories.tsx`** — State badge with workflow state variants
- **`beat-priority-badge.stories.tsx`** — Priority badge across all levels (0-4)
- **`beat-type-badge.stories.tsx`** — Type badge for work item categories
- **`command-palette.stories.tsx`** — Command palette open state
- **`Configure.mdx`** — Storybook configuration and intro page

## Assets

- `assets/` — Static images used in story decorators
