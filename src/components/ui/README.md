# ui

shadcn/ui primitives used throughout the Foolery interface.

## Key Files

- **`button.tsx`** — Button with variant and size props
- **`dialog.tsx`** — Modal dialog (overlay, content, header, footer)
- **`sheet.tsx`** — Slide-out side panel (used by settings)
- **`table.tsx`** — Table primitives (header, body, row, cell)
- **`tabs.tsx`** — Tab group with trigger and content panels
- **`command.tsx`** — Command menu (cmdk-based, used by command palette)
- **`dropdown-menu.tsx`** — Dropdown menu with items, separators, and sub-menus
- **`select.tsx`** — Select dropdown with trigger and content
- **`popover.tsx`** — Popover positioning container
- **`input.tsx`** — Styled text input
- **`textarea.tsx`** — Styled multiline text input
- **`badge.tsx`** — Inline badge with variant support
- **`card.tsx`** — Card container (header, content, footer)
- **`checkbox.tsx`** — Checkbox input
- **`label.tsx`** — Form label
- **`switch.tsx`** — Toggle switch
- **`separator.tsx`** — Visual divider
- **`tooltip.tsx`** — Hover tooltip
- **`alert-dialog.tsx`** — Confirmation dialog with cancel/action buttons
